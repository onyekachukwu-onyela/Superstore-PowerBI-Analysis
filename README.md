# Superstore Strategic Performance: From Loss Detection to Growth Strategy

### **Project Overview**
This 3-page interactive dashboard transforms raw retail data into a strategic roadmap. It moves beyond high-level KPIs to identify specific "profit bleeders" and prescribe actionable growth paths.

### **The 3-Page Story**
* **Executive Summary:** High-level overview of sales trends (2.30M) and seasonal Q4 spikes.
* **Product Profitability (The Deep Dive):** A diagnostic view that isolates **Tables** as the primary profit bleeder, largely attributed to aggressive discounting strategies.
* **Strategic Growth Opportunities (The Action):** An operational analysis identifying the **Consumer segment + Standard Class shipping** as the most efficient path to margin growth.

### **Tools Used**
* **Power BI:** Data visualization and UI/UX design.
* **Power Query:** Data cleaning and transformation.
* **DAX:** Calculated measures for Profit Margin and Quarterly evolution.
